# 🌐 Portfolio Personal | Adrián Durán Guillén (@SrAdryX)

Este repositorio contiene el código fuente de mi sitio web profesional, diseñado para centralizar mi trabajo como **IT & Full Stack Developer**. El objetivo es crear un entorno digital que refleje mi identidad técnica y mi capacidad para construir soluciones web desde cero.

---

## ⚡ El Proyecto
La web está concebida bajo una estética **Dark Minimalist**, optimizada para la lectura y enfocada en la experiencia de usuario técnica. 

* **Identidad:** Fusión de perfil profesional (Adrián Durán) y marca personal (@SrAdryX).
* **Tecnología:** Desarrollo nativo sin frameworks pesados para garantizar máxima velocidad y limpieza de código.
* **Enfoque:** Arquitectura limpia y diseño adaptable.

---

* **Estilos:** CSS3 con Custom Properties y animaciones dinámicas.
